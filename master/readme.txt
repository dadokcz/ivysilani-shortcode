Embed customized iVysilani.cz videos anywhere using shortcodes

== Description ==

A really simple and very lightweight plugin to embed customized iVysilani.cz videos anywhere in your blog or website using wordpress shortcodes.

= Showcase =

https://www.youtube.com/watch?v=5FS0beln3gs

= Functionality overview =

- Go to http://www.ceskatelevize.cz/ivysilani/
- Copy URL of Video
- Paste into editor via button or use shortcode [ivysilani url=http://www.ceskatelevize.cz/ivysilani/10396574515-milovnici-serialu/21238255364/ width=500 height=200]

== Installation ==

1. Upload `ivysilani-shortcode` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
